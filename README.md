一个简单有趣的贪吃蛇游戏，让你在闲暇时光享受经典游戏的乐趣。

## 游戏特点

- 经典贪吃蛇玩法
- 简单直观的操作
- 流畅的游戏体验
- 适合所有年龄段的玩家

## 游戏操作

- 使用方向键或WASD控制蛇的移动方向
- 吃到食物可以增加蛇的长度和分数
- 避免撞到墙壁或自己的身体
